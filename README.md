# 🔍 OnyForensic
The ultimate Windows forensic collection 80+ tools, PowerShell scripts, and analysis modules in a single place.

---

## 📦 What is OnyForensic?
OnyForensic is a desktop application that bundles, downloads, and organises the most popular digital forensic tools for Windows.
No more hunting for downloads everything is one click away, saved directly to Desktop.

---

##🛠️ Included Tools (80+)

WinPrefetchView, USBDeview, LastActivityView, TaskSchedulerView, BrowsingHistoryView, ShadowCopyView, DNSCacheView,BAM Parser, Prefetch Parser, PcaSvc, Process Parser, AmCacheParser, Timeline Explorer, SrumECmd, PECmd, ShellBags Explorer, INDXRipper, EvtxECmd, USBDeview Viewer, Easy Journal Viewer, Hayabusa, Chainsaw, Velociraptor, Process Hacker, EvtxExplorer, OSForensics, Alt Checker, Mod Analyser, Task Sentinel, Hex Viewer, Disk Drill, Recuva, TestDisk, Volatility, Rekall, DumpIt, MemProcFS and more.

---

## 🚀 Quick Start
Download the pre-built executable from Releases (if possible)

```bash
git clone https://github.com/OnyMouss/OnyForensic.git
cd OnyForensic
pip install -r requirements.txt
python app.py
open build.bat
Output: dist\OnyForensic.exe
First run creates a Checker folder on your Desktop where all tools are saved.
```
